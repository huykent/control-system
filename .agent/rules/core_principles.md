# Core Principles

- **Rule Evolution**: Mỗi khi phát hiện ra một rule nào quan trọng cần lưu lại để thực hiện sau này, hãy cảnh báo tới user và hỏi ý kiến xem có nên lưu rule này lại trong ".agent\rules\core_principles.md" không. Nếu user đồng ý thì hãy lưu.
- **Always Include Deployment Instructions**: Luôn viết hướng dẫn triển khai (deployment/implementation instructions) khi cung cấp giải pháp hoặc viết code.
